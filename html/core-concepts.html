Core Concepts in Programming

1. Pointers

Definition: A pointer stores the memory address of another variable.

Key Concepts

int *ptr;       // Declaration
int num = 10;
ptr = #    // Initialization
printf("%d", *ptr); // Dereferencing
Best Practices: Use free() after dynamic allocation.

2. Memory Management

int *arr = (int*)malloc(5 * sizeof(int)); // Heap allocation
free(arr); // Deallocation

Memory Segments

  • Stack: Local variables
  • Heap: Dynamic memory

3. Inbuilt C Libraries

  • stdio.h: printf(), scanf()
  • stdlib.h: malloc(), free()

4. Call by Value vs. Call by Reference

Call by ValueCall by Reference
Copies dataUses pointers
No side effectsModifies original data

5. Recursion

int factorial(int n) {
    if (n == 0) return 1;
    return n * factorial(n-1);
}

6. Iteration

int factorial(int n) {
    int result = 1;
    for (int i=1; i<=n; i++) 
        result *= i;
    return result;
}

Summary Cheat Sheet

Pointers: Use * and &.

Memory: Always free heap memory.

Recursion: Define a base case.
